# Olist E-Commerce Sales & Customer Analytics Project

## Project Overview

This project presents an end-to-end SQL and Power BI data analytics workflow using the Olist Brazilian E-Commerce dataset. The analysis focuses on customer behavior, sales performance, revenue trends, and RFM customer segmentation.

The project includes SQL-based data cleaning, customer analytics, KPI tracking, dashboard development, and business insights generation.

# Key Business Insights

* Loyal and Champion customers contributed significantly to overall revenue.

* Major sales contribution came from top-performing states.
* Customer purchase frequency varied significantly across segments.
